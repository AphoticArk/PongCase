All Libraries in this PixiJS Project Template downloaded via NPM January 2023:

- PixiJS v7.1.0
	- https://github.com/pixijs/pixijs
	- npm install pixi.js

- @pixi/graphics-extras v7.1.0
	- https://www.npmjs.com/package/@pixi/graphics-extras
	- npm install @pixi/graphics-extras

- GreenSock v3.x
	- https://greensock.com/docs/v3/Installation
	- npm install gsap

Note: 
- May want to switch to min versions (included in libs folder) or CDN versions for release

CDN Options:

- PixiJS:
    <script src="https://pixijs.download/release/pixi.js"></script>
    <script src="https://cdn.jsdelivr.net/npm/pixi.js@7.x/dist/pixi.min.js"></script>
    <script src="https://unpkg.com/pixi.js@7.x/dist/pixi.min.js"></script>

- GreenSock:
    <script src="https://cdnjs.cloudflare.com/ajax/libs/gsap/3.11.4/gsap.min.js"></script>
    <script src="https://cdnjs.cloudflare.com/ajax/libs/gsap/3.11.4/PixiPlugin.min.js"></script>
